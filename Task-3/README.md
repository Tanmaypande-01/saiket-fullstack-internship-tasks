# React To-Do List (Beginner Project)

## How to run

1. Install dependencies:

```bash
npm install
```

2. Start the dev server:

```bash
npm run dev
```

Vite will open the app in your browser.

## Features
- Add new task
- Edit task
- Delete task
- Mark task as completed

## Project structure
```
src/
  App.js
  App.css
  components/
    TodoForm.js
    TodoList.js
    TodoItem.js
```
